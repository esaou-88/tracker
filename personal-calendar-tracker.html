Integration notes

Google Calendar Best live-sync path: Google Calendar API with OAuth, then fetch events into this UI.
Proton Calendar Best practical path today: export ICS manually or use Proton Mail Bridge-compatible workflows if you later add a local sync helper.
Static HTML limitation Pure local HTML is ideal for layout and offline planning, but secure calendar login flows usually require a lightweight backend or trusted local app wrapper.
Suggested next build step
Add a small Node or Cloudflare Worker backend for Google OAuth and event sync, while keeping this HTML interface as the front end.
